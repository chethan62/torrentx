## What is TorrentX?

TorrentX is a native desktop GUI that connects to your local [Jackett](https://github.com/Jackett/Jackett) instance and searches all your configured indexers simultaneously. It compiles to a **single ~11 MB binary** with no runtime dependencies, starts instantly, and stays out of your way.

---

## Features

| Category | Details |
|----------|---------|
| **Search** | All Jackett indexers at once В· Deduplication across trackers В· Category filter В· Search history with per-item delete |
| **Results** | Sort by Seeders / Size / Date В· Filter by text, seeds, size, year, tracker, health В· Pagination (25 / 50 / 100 / All) |
| **Actions** | Open magnet В· Copy magnet В· Download `.torrent` В· Save to Favorites В· Open details in browser |
| **UI** | 19 themes (16 dark, 3 light) В· Adjustable row density В· Adjustable font size В· Detail side panel with ratio bar |
| **Extras** | Favorites with search & timestamps В· Export to CSV В· Animated search spinner В· Keyboard navigation |

### Keyboard Shortcuts

| Key | Action |
|-----|--------|

## Build from Source

### Requirements

- [Rust](https://rustup.rs/) (stable)
- Jackett running locally

### Install Rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```

### System Dependencies

<details>
<summary><b>Arch Linux</b></summary>

```bash
sudo pacman -S base-devel pkg-config openssl gtk3
```
</details>

<details>
<summary><b>Ubuntu / Debian</b></summary>

```bash
sudo apt install build-essential pkg-config libssl-dev libgtk-3-dev
```
</details>

<details>
<summary><b>Fedora</b></summary>

```bash
sudo dnf install gcc pkg-config openssl-devel gtk3-devel
```
</details>

<details>
<summary><b>Windows</b></summary>

Install [Rust for Windows](https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe) вҖ” Visual Studio C++ build tools are required ([setup guide](https://rust-lang.github.io/rustup/installation/windows-msvc.html)).
</details>

### Build & Run

```bash
git clone https://github.com/chethan62/torrentx
cd torrentx

# Development build
cargo run

# Optimized release build (~11 MB binary)
cargo build --release

# Run
./target/release/torrentx          # Linux / macOS
.\target\release\torrentx.exe      # Windows
```

> First build downloads and compiles dependencies вҖ” expect 2вҖ“4 minutes. Subsequent builds are fast.

## Usage

1. **Start Jackett** вҖ” [http://localhost:9117](http://localhost:9117)

2. **Open TorrentX** вҖ” a native window appears immediately.

3. **Configure** вҖ” click the вҡҷ Settings bar, paste your Jackett API key, adjust columns and timeout. Settings are saved automatically.

4. **Search** вҖ” type a query, pick a category, hit `Enter`. Results populate a sortable, filterable table.

5. **Act on results** вҖ” select a row and use keyboard shortcuts or the inline action buttons (`Mag` В· `Copy` В· `DL` В· `Fav` В· `Info` В· `Web`).
